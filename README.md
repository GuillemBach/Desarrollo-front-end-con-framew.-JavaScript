# Desarrollo-front-end-con-framew.-JavaScript
Login UOC: gbachsa
Guillem Bach Sayós
Descripción PEC1:
En esta primera PEC se ha realizado 2 tipos de actividades. La primera, la parte teórica, se han respondido varias preguntas relacionadas
con diferentes aspectos de los principales lenguajes de programación front end; HTML, CSS y JavaScript, y de herramientas de apoyo al dessarrollo. La segunda, la parte práctica, se han realizado 3 actividades diferentes, donde se ha propuesto una funcionalidad para la aplicación (iguales a las de los vídeos), añadiendole además un conjunto de funcionalidades extra; 
1.- La creación de un formulario de registro como la que podriamos encontrar en cualquier web. Añadiendo restricciones y requerimientos a la hora de completarlo.
2.- Aplicacción de conversión de valores de diferentes monedas, obteniendo el valor de estas a través de una API de terceros. Añadiendo las extrafuncionalidades de no poder seleccionar un valor negativo, indicar cuando el servidor está cargando (haciendo el proceso de consulta) y mostrar el error de consulta con el servidor.
3.- Aplicación para la reserva de asientos en un cine con elección de varias peliculas. Se le ha añadido la funcionalidad del conversor de valores de monedas para poder obtener el precio de las peliculas y de la sema de los asientos en la divisa elegida.
